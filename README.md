# stream-bet365-resource

## 项目简介

本仓库用于归档和发布多个独立 HTML 页面。这些页面均为静态资源，不针对任何特定域名或具体网站。仓库内容仅供学习、参考及个人归档使用。

## 目录说明

```
/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面所需的静态资源（CSS、JS、图片等）
└── README.md       # 本文件
```

- `pages/`：每个子文件夹或文件代表一个独立的页面归档。
- `assets/`：资源文件按需组织，与页面文件对应。

## 页面归档说明

- 每个页面均为独立完整的 HTML 文件，可直接在浏览器中打开。
- 页面内容不包含任何外部可执行代码或恶意脚本。
- 页面可能引用本地资源（位于 `assets/` 目录下），请确保完整克隆仓库以保持文件结构。
- 归档页面可能不定期更新或移除。
